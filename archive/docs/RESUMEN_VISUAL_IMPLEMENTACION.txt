║                    📝 NOTAS TÉCNICAS                                     ║
╚══════════════════════════════════════════════════════════════════════════╝

1. Logo en PDF:
   - Se carga dinámicamente con Image()
   - Se incrusta como imagen PNG
   - Tamaño: 20mm x 20mm
   - Posición: Esquina superior izquierda

2. Animaciones:
   - Usadas con CSS puro (sin librerías)
   - Optimizadas para navegadores modernos
   - Suave en todos los dispositivos

3. Responsive:
   - Funciona en mobile, tablet y desktop
   - Logo se adapta al tamaño de pantalla
   - Formulario centrado siempre

4. Accesibilidad:
   - Logo tiene alt text
   - Contraste de colores cumple WCAG
   - Funciona sin JavaScript habilitado
